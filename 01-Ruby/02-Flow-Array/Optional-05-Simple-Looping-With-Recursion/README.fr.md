## Contexte et objectifs

Il existe une troisième façon de créer des **boucles** avec un langage de programmation : avec la [récursion](http://stackoverflow.com/questions/6418017/what-is-ruby-recursion-and-how-does-it-work).

## Spécifications

- Écris une méthode en Ruby qui calcule la somme des entiers entre une valeur minimale et une valeur maximale (valeurs minimale et maximale comprises) en utilisant la **récursion**
- Si la valeur minimale est supérieure à la valeur maximale, elle doit retourner -1

## Astuce

Remarque que `1 + 2 + 3 + 4 == (1 + (2 + (3 + (4))))`
