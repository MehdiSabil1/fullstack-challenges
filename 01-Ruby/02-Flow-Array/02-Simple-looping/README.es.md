## Contexto y Objetivos

Un bucle en Ruby es una manera de ejecutar el mismo código varias veces. Es útil cuando quieres realizar la misma acción en una colección de elementos, o cuando quieres repetir una acción hasta que se cumpla cierta condición. Un buen ejemplo es si quieres preguntarle a un usuario si desea continuar haciendo algo, y quieres seguir preguntando hasta que diga “no”.

Aprendamos las dos maneras de hacer “loops” (bucles) en Ruby:

- Con un bucle `for`
- Con un bucle `while`

## Especificaciones

Escribirás dos funciones que hacen lo mismo, pero con sintaxis diferente:

- Escribe un método Ruby que compute la suma de enteros entre un valor mínimo y uno máximo (ambos incluidos).
- Si el valor mínimo es mayor al máximo, deberá devolver `-1`

`sum_with_for` debe usar un bucle `for`, mientras que `sum_with_while` debe usar un bucle `while`.
