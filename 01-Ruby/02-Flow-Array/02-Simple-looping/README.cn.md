## 背景与目标

Ruby中的循环是一种多次执行相同代码的方式。当你想对集合中的每个元素执行相同操作，或者想重复某个操作直到满足某个条件时，循环非常有用。一个好例子是：如果你想询问用户是否想继续做某件事，并且想一直询问直到他们说”不”。

让我们学习Ruby的两种”循环（loop）”方法：

- `for`循环
- `while`循环

## 详细说明

你将编写两个做同样事情的函数，但使用不同的语法：

- 用Ruby写一个方法（method），计算最小值和最大值之间的所有整数之和（包括最小值和最大值）。
- 如果最小值大于最大值，它应该返回`-1`。

`sum_with_for`应该使用`for`循环，而`sum_with_while`应该使用`while`循环。
